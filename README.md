# Seattle Godega Starter Site

This is a simple static website you can put online immediately.

## Files
- `index.html`
- `styles.css`

## What to change
In `index.html`, replace:
- `hello@seattlegodega.com` with your actual email if different.

## Easy hosting options
- GoDaddy website hosting
- Netlify
- Vercel
- GitHub Pages

For fastest launch: upload both files to your domain host or drag the folder into Netlify.
